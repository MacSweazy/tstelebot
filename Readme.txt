Описание телеграм-бота переводчика.


В основной дирректории имеется файл: "main.py"
Для запуска бота следует в командной строке набрать: "python main.py"


В процессе работы бота будет сформирован файл базы данных: "chat_bot/session.db", в котором будут храниться настройки языка для конкретных чатов, в которых присутствует бот.
Так же, в папке "utilities/config.ini" находится конфигурационный файл. В нём прописаны url - адреса и токены web-сервисов, с которыми работает бот.
В данном файле, в разделе: [tokens], следует вписать на место троеточия: 
уникальный ключ телеграм-бота, в "bot_token = ..."
и уникальный ключ api яндекс переводчика в "translate_token = ..."


Для завершения работы бота, в командной строке следует нажать комбинацию клавишь: ctrl + c.

Список команд бота:
/start - начало работы с ботом
/lang - выбор направления перевода
/help - справка
/tr[пробел]'текст' - перевести 'текст'. Пример:\n '/tr Привет мир!' "
/look посмотреть настройки языка


Для успешной работы требуется Python 3.6

Для установки зависимостей следует, воспользоваться файлом requirements.txt, для установки зависимостей через pip. 
Пример: 

"pip install -r requirements.txt"
